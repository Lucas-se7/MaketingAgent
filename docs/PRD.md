# 营销内容自动生成 Agent - 产品需求文档（MVP）

## 1. 产品定位

面向营销团队的智能化内容创作平台。**用户只需输入广告选题，Agent自主完成策划-生成-审核全流程，每步执行后询问用户是否采纳，用户确认后进入下一步，不采纳则根据意见重新执行。**

---

## 2. MVP功能范围

### 2.1 策划阶段

| 功能 | 描述 |
|------|------|
| 选题管理 | 创建、编辑、删除选题，设置优先级 |
| 品牌知识库 | 存储品牌故事、产品卖点，支持快速检索调用 |

### 2.2 生成阶段

| 功能 | 描述 |
|------|------|
| 内容生成 | 输入关键词/卖点/模板变量，生成营销文案 |
| 模板管理 | 创建、编辑、删除内容模板，支持变量占位符 `{品牌名}`、`{产品特点}` |
| 风格控制 | 支持正式/活泼/专业/幽默四种语言风格 |

### 2.3 审核阶段

| 功能 | 描述 |
|------|------|
| 质量检测 | 语法检查、敏感词检测、品牌调性校验（基于关键词匹配） |
| 审核流程 | 提交审核 → 待审核 → 通过 / 驳回（需填写理由） |
| 内容修改 | 驳回后可在原内容基础上修改，重新提交审核 |

---

## 3. 交互模式与工作流

**核心交互逻辑**：用户输入选题 → Agent自主执行当前步骤 → 询问用户是否采纳 → 用户确认则进入下一步 / 用户拒绝则根据意见重新执行


**内容状态**：
- **策划中**：Agent正在生成策划方案
- **待确认（策划）**：等待用户确认策划方案
- **生成中**：Agent正在生成内容
- **待确认（生成）**：等待用户确认内容
- **审核中**：Agent正在进行质量检测
- **待确认（审核）**：等待用户确认审核结果
- **已完成**：全流程通过

---

## 4. 用户角色

| 角色 | 职责 |
|------|------|
| 用户 | 输入选题，对每步结果进行采纳/不采纳确认，提供修改意见 |
| Agent | 自主完成策划、生成、审核全流程 |

---

## 5. MVP不包含的功能（后续迭代）

- 多平台适配
- 热点追踪
- 版本记录与对比
- 节日营销识别
- 多版本A/B生成
- 发布功能
- 管理员相关功能

---

## 6. 非功能需求

| 指标 | 要求 |
|------|------|
| 生成响应时间 | < 5秒 |
| 调性检测准确率 | > 85% |
| 敏感词覆盖率 | 基础敏感词库覆盖 |
